# MIUI-Patch-Gemini
- 支持MIUI9 7.8.24开发版。
- 因github单文件上传限制，MOD只提供了修改的部分，剩下的须手动合并。
- 已停止开发。
## 应用到非米5设备上
- 理论上去广告APK兼容Android N以上版本的MIUI系统（版本号一致），你可以直接替换。
- services.jar、boot.img不通用，请勿尝试，否则你的设备将无法启动。
